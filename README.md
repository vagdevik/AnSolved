# AnSolved

#### Analytics.py

Given a set of names, it finds the groups of matched names.

### How to run in terminal:
```
$ python Analytics.py
```
#### Sample Input:

Vladimir Frometa

Williams Bland

Vladimir A Frometa

Williams Bland Jr

#### Sample Output:

1. Vladimir Frometa
   
   Vladimir A Frometa


2. Williams Bland
   
   Williams Bland Jr
